# 2026-04-17 工作日志

## 优化 index.html / style.css / script.js

对苏州安驰电子科技有限公司官网首页进行全面代码优化：

### index.html
- 用 `<header>`, `<nav>`, `<section>`, `<article>`, `<footer>` 替换无语义 `<div>`
- 将3份重复内联SVG提取为 `<symbol>` + `<use>` 复用，HTML体积显著减小
- 合作伙伴图片全部加 `loading="lazy"` 懒加载
- 关于我们内容去掉 `<br>` 换行，改用 `<p>` 段落
- 侧边菜单 `.close-btn` 改为 `<button>` 标签，增加 aria 属性

### css/style.css
- 清理5处冗余 `white-space: nowrap`（只在 `.nav` 声明一次，子元素继承）
- 右侧悬浮栏从无意义命名（`hdhda0011`, `jsjkdjs`, `sdkjf101`, `kdjjdf`, `ha4s1`）改为 BEM 风格（`.float-bar`, `.float-popup__item` 等）
- 增加 `.nav-list li a.active` 导航激活样式
- 产品/优势卡片改用 `flex: 1 1 calc(33.333% - 14px)` + `gap`，去除硬编码 `margin-bottom`
- 产品卡片增加 `:hover` 微动效

### js/script.js
- 选择器改用语义化新类名
- 提取 `openSidebar` / `closeSidebar` 命名函数，逻辑更清晰
- 增加键盘方向键控制轮播图
- 轮播切换同步更新 `aria-selected` 属性
